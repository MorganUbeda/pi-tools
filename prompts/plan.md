# Planning Session

## Goal
Create an implementation plan through conversation. Do NOT write any files.

$@

## Workflow
1. **Understand design** - Review what needs to be implemented
2. **Break down** - Divide into small, independent tasks
3. **Define criteria** - Clear verification for each task
4. **Gather context** - Identify all relevant information
5. **Iterate** - Refine through discussion

## Task Structure
Each task needs: **What**, **How**, **Verify**, **Context**, **Dependencies**

## Rules
- Break into small, independent tasks with clear completion criteria
- Ask user when intent is unclear; search online for missing info
- Never make design decisions autonomously
- Ask user on blockers; don't stop planning
- **Never write files or implement; only plan**

**STOP** when plan is complete; wait for instruction to write it.
